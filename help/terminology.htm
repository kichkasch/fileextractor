Terminlogy

What are all these expressions about?

FileTypes
Files on computers are stored in different structures. (for example word documents, images, text documents, movies, ...). In Windows Operating Systems these are usually indicated by their file extension (doc, jpg, bmp, wav, mp3).
Image
Exact binary copy of your (corrupt) data source
Imaging
The process of creating the exact copy (image) of your source
Core (ImageGenerator)
The process of Imaging is performed on a very low level within the operating system (OS) and therefore needs to be implemented for each OS seperately. Each implementation is called a core - choose the one from the drop-down box, which is closest to your OS.
ImageGenerator
A module of FileExtractor for creating the image of your data source.
Operating System (OS)
The interface between the user / applications and the hardware. Most commonly used these days Windows or Linux.
Output directory
Folder in your local filesystem, where you recovered files will be stored
