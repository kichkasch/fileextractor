Signatures

Overview

In order to provide a mechanism for integrating lots of file types with the program, a signature approach has been implemented; hence, all one need to do in order to extend the program to support an additional file type is writing a new signature for it.

All information about signatures is and must be provided in the Python source file signatures.py. It is located directly in the program folder.

Activate and Deactivate Signatures

There might be a need to deactivate certain file types for a execution of the program. On the one hand, it might speed up the execution a bit, on the other hand there might be files with very short sequences which might cause lots of false positives and there are not really needed for the execution.

By default, all signatures are activated. You may deactivate them by unticking them in the list of signatures in the start screen of the application.

The 3 Types in detail
Start sequence - End sequence
Start sequence - End file information inside file
Start sequence - Manual file end determination
